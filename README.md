# Chatbot
ChatBot ini berbasis rag berdasarkan dokumen pdf, excel, docx, txt, json

Fitur yg tersedia:

Chatbot RAG Segmentasi : ex: Divisi Finance, IT, HR, ETC (Jawaban hanya berdasarkan dokumen yg pernah diupload dan tersimpan di dalam vector database seperti milvus atau oracle 23ai)
Chatbot Summarization : Menyimpulkan dokumen yang diupload ke dalam chat, untuk menampilkan point terpenting saja
Chatbot Assistant : Bot assitant universal yang menjawab berdasarkan data yang tersedia pada modal
Pemilihan model AI yang tersedia atau sudah terinstall pada local : User dapat memilih model chatbot yang ingin mereka gunakan seperti contohnya llama3 atau mistral
